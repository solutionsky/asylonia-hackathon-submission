# Asylonia Running Evidence

Live demo:
https://asylonia.com

Evidence included:
- Firebase Authentication works as Asylonia Passport
- Credit balance appears in the header
- Pricing page shows credit packages, Civic Reservation, and SVC Recorder
- AI Creation Hall music flow is implemented
- My Vault loads user-owned assets from Firestore
- Civic Reservation creates a Vault asset
- SVC Showcase Recorder can be opened from My Vault and records locally through browser permission
